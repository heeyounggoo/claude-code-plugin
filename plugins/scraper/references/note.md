# 통일 노트 포맷

모든 스킬은 아래 포맷으로 노트를 작성한다.

---

## 노트 템플릿

```markdown
# {제목}

| 항목 | 내용     |
| ---- | -------- |
| 날짜 | {DATE}   |
| 출처 | {SOURCE} |
| 태그 | {TAGS}   |

{CONTENT}

## 요약

**주제:** {SUBJECT}

**인사이트**
- {인사이트 1}
- {인사이트 2}
...
```

---

## 변수 정의

| 변수        | 설명                                                                 |
| ----------- | -------------------------------------------------------------------- |
| `{제목}`    | 콘텐츠 제목 (영상 제목 / 아티클 제목 / 파일명)                       |
| `{DATE}`    | config.md의 DATE 변수                                                |
| `{SOURCE}`  | YouTube URL / 웹 URL / 로컬 파일 전체 경로                           |
| `{TAGS}`    | 내용 주제 태그 + 스킬 고유 태그 (`#youtube` / `#article` / `#file`) |
| `{CONTENT}` | 각 스킬이 채우는 본문 — 스킬별 구조는 각 SKILL.md에 기술             |
| `{SUBJECT}` | 핵심 주제 2-3줄 요약                                                 |

---

## 스킬별 {CONTENT} 구조

### youtube
```markdown
## 주요 내용

{영상 흐름에 따라 섹션별 정리}

### {섹션 제목}
- {핵심 포인트}
```

### article
```markdown
## 주요 내용

{내용 흐름에 따라 섹션별 정리. 다른 언어는 한국어로 번역.}

### {섹션 제목}
- {핵심 포인트}
```

### file
```markdown
## 주요 내용

{파일 유형에 맞는 분석 결과 섹션별 정리}

### {섹션 제목}
- {핵심 포인트}
```
